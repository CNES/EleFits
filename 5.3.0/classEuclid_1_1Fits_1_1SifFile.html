#include <SifFile.h>

Detailed Description

Single image FITS file handler.

Through constructor and destructor, SifFile handles creation, opening, closing and removing of the SIF-file. Other methods give access to the header unit and image data unit. Note that the data unit is empty at creation, and must be resized to write data.
