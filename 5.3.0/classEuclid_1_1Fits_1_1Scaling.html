Scaling Class Reference

#include <Scaling.h>

Detailed Description

Helper class for scaling parameters.

A scaling can be:

  • Disabled (null value);
  • Provided as an absolute value;
  • Provided as a relative value, either as a multiplicative factor or its inverse.
