ColumnKey Class Reference

#include <ColumnKey.h>

Detailed Description

A lazy-evaluated column key initialized either with an index or a name.

If missing at the time of usage, the index or name is evaluated thanks to a BintableColumns object. Negative (backward) indices are also resolved lazily. After evaluation, the value is cached.

Note
Constructors are intentionally not explicit.
