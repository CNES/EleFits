Class Description

template<typename TUnit>
class Euclid::Fits::Test::Chronometer< TUnit >

A simple chronometer with increment times and elapsed time caching.

Template Parameters
TUnitThe time unit, e.g. std::chrono::milliseconds

Each time the chronometer is started and stoped, an increment is computed, and the total elapsed time is incremented. An offset can be provided, which is the initial value of the elapsed time, but has no effect on the increments.

Simple statistics on the increments can be computed (e.g. mean increment).

The chronometer can be reset, which means that the list of increments is emptied, and the elapsed time is set to 0 or the offset.
